To edit the webage text:

    Go to the repo ( https://github.com/DeeNewcum/termdetect ),
    click "Settings" (on the right),
    and click the "Automatic page generator" button.

    After you're done editing, choose the "Midnight" theme.

Then run these commands:

    cd $(git rev-parse --show-toplevel)
    git checkout gh-pages
    git pull origin
    ./fix_autogenerated
    git add --all
    git commit -m 'fix autogenerated'
    git push origin
    git checkout master
    xdg-open http://deenewcum.github.io/termdetect/     # manually confirm it looks good
